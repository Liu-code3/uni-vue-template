# uni-cli + vite + vue

# 项目运行

```shell
git clone https://github.com/Liu-code3/uni-vue-template.git
cd uni-vue-template (进入项目)
pnpm install (安装依赖包)
pnpm run dev:mp-weixin (运行项目)
```
